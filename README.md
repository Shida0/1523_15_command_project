# 1523_15_command_project
Это репозиторий проекта по теме "Астероидная опасность"

На данный момент приложение:
  - Сохраняет данныне обо всех достаточно больших и известных астероидах, в том числе и потенциально опасных, в базу данных
  - Оттуда берутся данные обо всех потенциально опасных астероидах
  - Мы проверяем есть ли возможность сближения с Землей ближе чем на 7.5 млн км. Это происходит благодаря расчетам эфемерид.
  - Мы проверяем диаметр астероидов - либо из наших данных, либо по альбедо 0.15. Мы в БД будем учитывать, какое альбедо мы брали - изначальное или 0.15
  - Таким образом расчитывается потенциальная опасность астероида

Сближения и опасность для каждого астероида записываются в БД. Каждый день в 00:00 данные обновляются.

Нужно сделать:
  - Контроллеры для управления данными из БД
  - Сервисы для бизнес логики
  - API эндпоинты
  - Фронтенд

Для поиска и мониторинга астероидов используется библиотека astroquery.
База данных - PostgreSQL, библиотеки - asyncpg и sqlalchemy, миграция бд - alembic.
