# 1523_15_command_project
Это репозиторий проекта по теме "Астероидная опасность"

На данный момент приложение:
  - Получает данные обо всех астероидах, орбиты которых теоретически могут сближаться с Землей
  - Использует критерий: перигелий ≤ 1.3 а.е. (астероид приближается к орбите Земли)
  - Это постоянный каталог объектов, которые в принципе могут быть опасны

В дальнейшем надо будет добавить мониторинг текущих сближений.

Для поиска астероидов используется библиотека astroquery. Данные сохраняются в json и csv файлы.
