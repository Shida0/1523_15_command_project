# 1523_15_command_project
Это репозиторий проекта по теме "Астероидная опасность"

На данный момент приложение:
  - Сохраняет данныне обо всех достаточно больших и известных астероидах, в том числе и потенциально опасных, в csv и json файл
  - Оттуда берутся данные обо всех потенциально опасных астероидах
  - Мы проверяем есть ли возможность сближения с Землей ближе чем на 7.5 млн км в следущие 30 дней. Это происходит благодаря расчетам эфемерид.

В дальнейшем надо будет добавить базу данных вместо csv.

Для поиска и мониторинга астероидов используется библиотека astroquery. Данные сохраняются в json и csv файлы.
