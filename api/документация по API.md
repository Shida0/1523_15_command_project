# 🌌 API документация Asteroid Watch

> **Для кого этот документ:** Эта документация написана так, чтобы даже человек без опыта программирования мог понять, как работать с нашим API.

---

## 📖 Что такое API?

**API** (Application Programming Interface) — это способ одной программы общаться с другой. 

**Простая аналогия:** Представьте ресторан:
- **Вы (клиент)** = наше приложение или сайт
- **Официант** = API
- **Кухня** = сервер с базой данных

Вы делаете заказ (запрос) → официант несёт его на кухню → кухня готовит → официант приносит блюдо (ответ).

## 📡 Базовые понятия

### URL запроса

Каждый запрос имеет адрес:
```
http://localhost:8000/asteroids/all?skip=0&limit=10
│                        │         │
│                        │         └─ Параметры запроса
│                        └─ Путь (endpoint)
└─ Адрес сервера
```

### Методы запросов

Мы используем **GET** — это значит «получить данные».

### Параметры запроса

Некоторые запросы принимают параметры для уточнения:

| Параметр | Тип | Что означает | Пример |
|----------|-----|--------------|--------|
| `skip` | число | Пропустить N записей | `skip=20` — пропустить первые 20 |
| `limit` | число | Вернуть не более N записей | `limit=10` — максимум 10 штук |
| `max_moid` | число | Максимальное расстояние | `max_moid=0.05` — только близкие |

**Пример:**
```bash
# Пропустить 20 астероидов, показать следующие 10
curl "http://localhost:8000/asteroids/all?skip=20&limit=10"
```

## 🪨 Астероиды (`/asteroids`)

### 1. Все астероиды

**Что делает:** Возвращает список всех астероидов из базы.

**Запрос:**
```bash
curl "http://localhost:8000/asteroids/all?skip=0&limit=5"
```

**Ответ (упрощённо):**
```json
[
  {
    "id": 1,
    "designation": "433",
    "name": "Eros",
    "estimated_diameter_km": 16.84,
    "earth_moid_au": 0.028,
    "orbit_class": "Amor"
  }
]
```

---

### 2. Околоземные астероиды

**Что делает:** Возвращает астероиды, которые подлетают близко к Земле.

**Параметр:** `max_moid` — максимальное расстояние в а.е. (астрономических единицах)

**Запрос:**
```bash
# Астероиды ближе чем 0.03 а.е.
curl "http://localhost:8000/asteroids/near-earth?max_moid=0.03&limit=5"
```

---

### 3. Астероиды по типу орбиты

**Что делает:** Возвращает астероиды определённого класса.

**Классы орбит:**
- `Apollo` — пересекают орбиту Земли
- `Aten` — тоже пересекают, но меньшие
- `Amor` — летают между Землёй и Марсом

**Запрос:**
```bash
# Только астероиды класса Apollo
curl "http://localhost:8000/asteroids/orbit-class/Apollo?limit=5"
```

---

### 4. Статистика по астероидам

**Что делает:** Возвращает общую статистику.

**Запрос:**
```bash
curl "http://localhost:8000/asteroids/statistics"
```

**Ответ:**
```json
{
  "total_asteroids": 2547,           # Сколько всего
  "average_diameter_km": 0.85,       # Средний размер
  "min_earth_moid_au": 0.0002,       # Самый близкий
  "accurate_diameter_count": 320     # С известным диаметром
}
```

---

### 5. Подробно об одном астероиде

**Что делает:** Возвращает всё, что мы знаем об астероиде.

**Запрос:**
```bash
# Астероид 433 (Эрот)
curl "http://localhost:8000/asteroids/433"
```

**Ответ:**
```json
{
  "designation": "433",
  "name": "Eros",
  "estimated_diameter_km": 16.84,
  "orbit_class": "Amor",
  "close_approaches": [...],      # Все сближения
  "threat_assessment": {...}       # Оценка опасности
}
```

## 📘 Словарь терминов

### 📏 Единицы измерения

| Единица | Что это | Пример |
|---------|---------|--------|
| **а.е.** (AU) | Астрономическая единица = расстояние от Земли до Солнца | 1 а.е. ≈ 150 млн км |
| **км/с** | Километров в секунду | 20 км/с = 72 000 км/ч |
| **Мт** | Мегатонна = 1 миллион тонн тротила | 100 Мт ≈ 2000 атомных бомб |

### 🌡️ Шкалы опасности

#### Туринская шкала (0-10)
| Уровень | Цвет | Значение |
|---------|------|----------|
| 0 | 🟢 | Нет риска |
| 1-3 | 🟡 | Низкий риск, нужно наблюдать |
| 4-7 | 🔴 | Серьёзная угроза |
| 8-10 | 🟣 | Неизбежное столкновение |

#### Палермская шкала
- **Отрицательная** (например, -2.5) — риск ниже обычного фона
- **0** — риск равен фоновому
- **Положительная** — риск выше фона

### 🛰️ Типы орбит

| Тип | Описание |
|-----|----------|
| **Apollo** | Орбита пересекает земную, большая полуось > 1 а.е. |
| **Aten** | Орбита пересекает земную, большая полуось < 1 а.е. |
| **Amor** | Орбита между Землёй и Марсом, не пересекает земную |

## ❓ Частые вопросы

### Почему некоторые числа с плавающей точкой?
Потому что в космосе всё измеряется очень точно. 0.028 а.е. — это примерно 4.2 миллиона километров.

### Что такое `designation`?
Это официальное обозначение астероида в системе NASA. Например, `433` или `2023 DW`.

### Почему некоторые астероиды без имени?
У многих астероидов есть только номер. Имя дают только после подтверждения орбиты.

### Как часто обновляются данные?
Раз в 24 часа автоматически из NASA API.

## 🆘 Если что-то не работает

### Ошибка 404
**Что значит:** Объект не найден

**Пример:**
```bash
curl "http://localhost:8000/asteroids/несуществующий"
# Ответ: {"detail": "Asteroid 'несуществующий' not found"}
```

**Решение:** Проверьте обозначение астероида

### Ошибка 500
**Что значит:** Проблема на сервере

**Решение:** Перезапустите сервер

### Сервер не отвечает
**Проверьте:**
1. Сервер запущен (`python main.py`)
2. Порт 8000 не занят другой программой
3. Вы в правильной папке проекта
