Защита Земли от астероидной опасности

Asteroid Watch — это комплексная система мониторинга потенциально опасных астероидов (PHA), которая отслеживает околоземные объекты, анализирует их орбиты и оценивает риск столкновения с Землёй. Мы используем данные NASA Sentry API для оценок угроз и NASA CAD API для сближений.

Важно: только PHA астероиды

В нашей базе данных содержатся только потенциально опасные астероиды (PHA). Это наиболее опасные околоземные объекты, требующие постоянного мониторинга.

Почему это важно?

Астероидная опасность — реальная угроза, которую необходимо отслеживать и изучать

☄️

Исторические прецеденты

65 миллионов лет назад астероид диаметром ~10 км привёл к вымиранию динозавров. В 1908 году Тунгусский метеорит разрушил 2000 км² тайги. В 2013 году Челябинский метеорит повредил здания и ранил 1500 человек.

🎯

Потенциальная опасность

В настоящее время известно более 32,000 околоземных астероидов. Из них около 2,400 классифицируются как потенциально опасные (PHA). Даже астероид диаметром 140 метров может вызвать региональную катастрофу.

🔬

Научное значение

Изучение астероидов помогает понять происхождение Солнечной системы, состав первичного вещества, а также оценить ресурсы для будущего космического освоения. Многие астероиды содержат ценные металлы.

🛡️

Защита планеты

Раннее обнаружение опасных объектов даёт время для разработки миссий по отклонению астероида. Миссия DART NASA в 2022 году успешно продемонстрировала возможность изменения орбиты астероида.

📡

Источник данных об угрозах

Все оценки угроз столкновений (Туринская шкала, Палермская шкала, вероятность удара) получены из NASA Sentry API. Наша система не рассчитывает угрозы самостоятельно — мы агрегируем, переводим и визуализируем официальные данные NASA для удобного мониторинга.

Туринская шкала опасности

Международная шкала для оценки опасности столкновения астероидов с Землёй (данные NASA Sentry)

0Нет риска
1Нормальный
2Внимание
3Угроза
4Высокий
5Серьёзный
6Угроза
7Опасный
8Катастрофа
9Глобальный
10Коллизия
Уровень 0 (Зелёный) Вероятность столкновения практически нулевая, или объект слишком мал.
Уровень 1 (Зелёный) Обычное открытие. Вероятность столкновения очень низкая.
Уровень 2-4 (Жёлтый/Оранжевый) Требует внимания астрономов. Вероятность столкновения 1% или выше.
Уровень 5-7 (Красный) Серьёзная угроза. Вероятность столкновения значительная.
Уровень 8-10 (Бордовый) Неизбежное столкновение с локальными или глобальными последствиями.
