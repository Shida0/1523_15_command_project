
### Форматирование кода

``` bash
# Автоматическое форматирование
black .

# Проверка стиля
flake8

# Сортировка импортов
isort .
```

### Миграции базы данных

``` bash
# Создание новой миграции
alembic revision --autogenerate -m "Описание изменений"

# Применение миграций
alembic upgrade head

# Откат миграции
alembic downgrade -1

# Просмотр истории миграций
alembic history
```
